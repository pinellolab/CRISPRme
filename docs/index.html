CRISPRme

Variant-aware CRISPR off-target nomination

Hosted instance retired

The hosted web application previously served at crisprme.di.univr.it is no longer available, as institutional hosting at the University of Verona is no longer available for this service. CRISPRme itself remains fully open-source and actively maintained. The full web interface can be run locally on your own machine — with the same functionality as the previous hosted version — using the instructions below.

Run the CRISPRme web interface locally

The local web interface is equivalent to the retired hosted version. Choose either Conda/Mamba or Docker.

Option 1 — Conda / Mamba

mamba create -n crisprme python=3.9 crisprme -y
mamba activate crisprme
crisprme.py --version
crisprme.py web-interface

Option 2 — Docker

docker pull pinellolab/crisprme
docker run -v ${PWD}:/DATA -w /DATA -i pinellolab/crisprme crisprme.py web-interface

Then open the URL printed in your terminal in a browser (Chrome, Firefox, or Safari).

Quick test

Verify the installation with a small-scale test on chromosome 22 using the 1000 Genomes dataset:

crisprme.py complete-test --chrom chr22 --vcf_dataset 1000G
